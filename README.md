## Step up target Ubuntu

Install Nodejs 16.x LTS

* sudo apt install openssl wget ffmpeg

* wget https://dl.strem.io/four/master/server.js
* wget https://dl.strem.io/four/master/stremio.asar

* chmod +x start.sh

* ./start.sh
